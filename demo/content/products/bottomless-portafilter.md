# 58mm Bottomless Portafilter

A bottomless (naked) portafilter removes the spouts so you can watch the shot pour directly from the
basket. It's the best diagnostic tool for [channeling](/brewcraft/docs/troubleshooting): even, mirror-like flow
means good [puck prep](/brewcraft/docs/puck-prep); spraying or squirting reveals uneven extraction. Fits all 58mm
BrewCraft machines except the entry [Pico](/brewcraft/products/pico-machine), which ships a pressurized basket
system. Note: a bad puck will spray coffee, so expect some mess while you learn.
