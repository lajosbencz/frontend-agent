# BrewCraft Hand Grinder

A hand-cranked grinder with 38mm stainless conical burrs and a stepless adjustment collar. It grinds fine
enough for espresso and is a genuinely good starter option — the main cost is effort (an espresso dose
takes 30-45 seconds of cranking).

- Burr type: conical, 38mm stainless
- Adjustment: stepless (numbered collar for repeatability)
- Capacity: single dose (~30g)
- Power: manual

Best paired with the [Pico](/brewcraft/products/pico-machine) or [Solo](/brewcraft/products/solo-machine) as a first setup.
If you're grinding for espresso daily, an electric grinder like the [Compact](/brewcraft/products/compact-grinder)
will save your arm. See the [grinder buying guide](/brewcraft/docs/buying-guide-grinders) and
[grinder burrs](/brewcraft/docs/grinder-burrs) explainer.
